瑶族 - 瑶族与畲族

一脉相承

摇，本意是制作陶器的一项工艺，有旋转、回旋、盘旋的意思。陶、窑、吴回、盘瓠、摇民、盘龙乃是一脉相承的。

瑶族的始祖名叫“盘瓠”，是上古姚姓有虞氏的族人。帝高辛时期，有虞氏首领吴回担任华夏部落联盟火正祝融之职。犬戎入侵中原，吴回领兵征讨，同族的盘瓠取得犬戎首领的首级，立下大功，帝高辛于是妻之以女，封土赐姓命氏，在易水流域建立勾姓有易氏因民国。

上古时期

帝舜时期，帝舜之子戏被封于淮阳，建立妫姓有庳氏摇民国。

夏代初期，有易氏、有虞氏曾经帮助夏王少康复国，勾姓有易氏是少康的母族、姚姓有虞氏是少康的妻族，有易氏、有虞氏都是夏王朝的同盟国，世代联姻。

夏代中期，商族崛起于有易氏东北的燕山地区，有易氏是商族的母族，一度关系非常好。但是，商族的崛起也给有易氏带来许多不利因素，故有易氏之君绵（黾）臣挑起争端，杀了前来跨国与黄河中下游河伯国做牛羊买卖的商族首领子亥，但是放走了子亥的弟弟恒，从而埋下了灭国的种子。

商族在恒、相土、曹圉、上甲微等首领几代经营之后，势力强大，扩张到了山东半岛、黄河南岸，对有易氏形成了包围。上甲微借道河伯国，对有易氏南北夹攻，有易氏灭亡。后裔一部在河伯国的帮助下逃到南方投奔同族姚姓有虞氏和妫姓有庳氏。

夏代后期，勾姓有易氏余部同妫姓有庳氏一部在河南淮阳建立摇民国，祭祀盘瓠和戏。

商代前期，有虞氏三部受到商王朝压迫，部族南迁越过桐柏山和大别山，来到帝舜南征埋骨之地“苍梧山”（即湖北大悟山，又名天吴山），在此建立多个小国，其中就有武汉北面盘龙城的“盘龙”国、武汉东面阳逻镇香炉山的“戈水”国等。

商代中后期，有虞氏被称为“虎方”，武丁大力南征。有虞氏三部再次南迁到长江南岸，在鄱阳湖以西、洞庭湖以东的现新干县吴城遗址建立南“吴国”，这个南吴国也被商王朝称为“虎方”。

西周时期，“淮夷”的后裔楚国崛起于江汉流域，也多次南征“南吴国”。同时，西周王朝的同族原本位于今枣庄地区的姬姓东吴国在西周王朝的支持下向南扩张，掠取了淮夷诸国的大片领地；在春秋中期，姬姓东吴国势力扩张到长江下游南岸，成为南吴国的又一个劲敌。同时期，受到东吴国压迫的淮夷一部南迁到浙江地区，在此得以喘息，建立了越国。南吴国的西面的武陵地区，乃是最初吴回、帝舜等大力征讨的三苗族。

四面都是强敌，所以南吴国最终在春秋末期支离崩析，四分五裂；大部国土被楚、东吴、越吞并，一部加入苗族，成为“武陵蛮”。余部南迁，越过南岭到达两广，成为现瑶族、畲族的前身。

瑶族与畲族

瑶族之中最古老的部族是信奉“盘瓠”的盘瑶和畲瑶。盘瑶又被称板瑶、勉瑶，盘瑶自称“棉”或“勉”，即是人的意思，应该源于“有易之君绵臣”之“绵”、“黾”，黾也有龙的形状；畲族乃是瑶族的别部畲瑶，也是最原始的一支，一直保持“刀耕火种”的生活习惯，在南宋时期单独成为部族。

畲瑶的四大姓（盘、蓝、雷、钟）都与有虞氏有关：盘姓源于盘瓠，盘是一种器皿，本义是摇；雷姓源于雷神吴回，吴回也叫吴雷；钟姓源于鐘，鐘也是一种铜器；蓝源于监，监即鉴，即铜镜，引申为监督。

瑶族之中有盘瑶十二姓，盘、沈、冯、黄、李、邓、蒋、雷、唐、胡、周、赵，与畲族有相同的盘、雷二姓，广西、福建的瑶族中蓝姓、钟姓也是大姓。

至新中国成立之后，这个古老而历尽磨难的部族有了正式的族名“瑶族”、“畲族”。
